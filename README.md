# Sticky-Notes

## Description

Posti-it notes made in React

## Installation

Download the files and install npm.

```bash
npm i
```

Start local server
```bash
npm start
```


type http://localhost:3000/ in your browser adress bar


## Usage

Create sticky notes so you dont forget something.
